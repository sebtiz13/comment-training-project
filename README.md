# Comment training project

Create a form in React that calls an API (symfony + JWT). The API push message to RabbitMQ,
this message is depiled by an symfony cli worker to send email.

## Stack

- Symfony 5
- JWT Authentication
- RabbitMQ
- React
- Docker
- Mercure

## Tools

- PHP CS Fixer
- PHPStan
- ESlint

## Languages

- PHP 8.0
- Typescript 4.1
